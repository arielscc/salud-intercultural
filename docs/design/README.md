# Design

Documentacion del sistema visual y criterios de experiencia de usuario. Este directorio define como debe verse, sentirse y comportarse la interfaz publica.

Usar este directorio para:

- Paleta, tipografia, espaciado, componentes visuales y patrones UI.
- Reglas de consistencia visual para nuevas paginas o componentes.
- Criterios de experiencia que deben mantenerse al evolucionar a V3.

No usar este directorio para QA operativo, performance, accesibilidad tecnica o media administrable; eso vive en `docs/operations`.
