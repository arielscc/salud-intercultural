# Features

Dominios de producto de la V2. Cada feature puede agrupar componentes, schemas,
acciones, servicios y helpers propios.

Estructura recomendada por feature:

```txt
feature-name/
├── components/
├── schemas/
├── services/
├── actions/
└── index.ts
```

Features previstas:

```txt
home
services
treatments
team
testimonials
faq
contact
leads
admin
seo
analytics
```
