# shadcn/ui

Directorio reservado para componentes generados por shadcn/ui.

La configuracion vive en `components.json` y apunta a:

```txt
@/components/ui
@/lib/utils
@/hooks
```

Los componentes existentes de la landing permanecen en `src/components/shared` y
`src/components/landing` hasta que una tarea futura haga la migracion visual.
